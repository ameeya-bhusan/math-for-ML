# Mathematics for Machine Learning

This repository documents my completion of the "Mathematics for Machine Learning" specialization on Coursera, with structured notes and applications of core mathematical concepts used in machine learning.

## 📄 Certificate
- [View Certificate](certificates/math-for-ml.pdf)

## 📚 Topics Covered

### 1. Linear Algebra
- Vectors and matrices
- Eigenvalues and eigenvectors
- Matrix decompositions
- Basis transformations

### 2. Multivariable Calculus
- Partial derivatives
- Gradients and Jacobians
- Optimization and critical points
- Chain rule in multiple dimensions

### 3. Probability & Statistics
- Probability distributions
- Expectation and variance
- Bayesian thinking
- Sampling and estimation

## 🧠 Key Learnings

- Developed a strong mathematical foundation for machine learning algorithms
- Understood how linear algebra governs data representation and transformations
- Learned how gradients drive optimization in ML models
- Built intuition for probabilistic reasoning in data-driven models

As a PhD researcher working on:

This course is directly relevant:
